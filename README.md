# Анализ скважин A15, A16, A10

## Описание
Автоматический петрофизический анализ LAS файлов скважин.

## Что делает
- Загрузка и очистка данных
- Построение каротажного планшета
- Корреляционный анализ параметров
- Определение потенциальных коллекторов

## Данные
Скважины A15, A16, A10 из месторождения [Kaggle](https://www.kaggle.com/datasets/sahasourav17/well-logs)

## Технологии
Python, Lasio, Pandas, Matplotlib, Seaborn

## Результаты
- A15: 10 интервалов коллектора
- A16: 4 интервала коллектора
- A10: 18 интервалов коллектора
